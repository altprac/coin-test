#  Task

## Installation
 
### Server

cd server

npm install

npm start *(start server on localhost:3000)*

### Client

cd client

npm install 

*for dev*

 npm run dev *(start client on localhost:8080)*

 *for prod*
 
 npm run build *(build client to the dist folder on the server which can be accessed @ localhost:3000)*
